TARGET PROFIT & LOSS
Fermano l'intera strategia al raggiungimento dell'obiettivo giornaliero
richiedono Fascia Oraria attivaStop the entire strategy when the daily target is reached
require active Time Window
💰 TARGET PROFIT💰 TARGET PROFIT
Il BOT calcola il profitto NETTO (guadagni - perdite - commissioni).
Quando raggiunge il valore impostato: chiude i trade aperti, mostra etichetta e ferma qualsiasi nuovo segnale fino al giorno successivo.The BOT calculates NET profit (gains - losses - commissions).
When it reaches the set value: closes open trades, shows a label and stops any new signal until the next day.
📌 Target = 500€ → alle 17:42 il netto supera 500€ → tutto si ferma per oggi📌 Target = 500€ → at 17:42 net exceeds 500€ → everything stops for today
💸 TARGET LOSS
Il BOT calcola la perdita NETTA della sessione.
Quando la perdita netta supera il valore: chiude i trade aperti, mostra etichetta e ferma qualsiasi nuovo segnale per proteggere il capitale.The BOT calculates the NET loss of the session.
When net loss exceeds the value: closes open trades, shows a label and stops any new signal to protect capital.
📌 Target Loss = 300€ → alle 16:15 la perdita netta tocca -300€ → tutto si ferma per oggi📌 Target Loss = 300€ → at 16:15 net loss hits -300€ → everything stops for today
⏰ Richiedono Fascia Oraria OBBLIGATORIAMENTE attiva · senza fascia il sistema si blocca con erroreRequire Time Window MANDATORILY active · without it the system stops with an error
🔀 Se entrambi attivi → il sistema reagisce al PRIMO evento raggiunto (profit o loss)If both active → the system reacts to the FIRST event reached (profit or loss)
📅 Si resettano automaticamente ogni giorno alla mezzanotteAutomatically reset every day at midnight
🧮 Il calcolo è NETTO: profitti lordi - perdite lorde - commissioni totali della sessioneThe calculation is NET: gross profits - gross losses - total commissions of the session

FUSO ORARIO
Impostazione del fuso orario in base al broker e alla stagione
fondamentale per allineare correttamente la Fascia OrariaTime zone setting based on broker and season
essential to correctly align the Time Window
Il parametro Fuso Orario indica lo scarto in ore tra l'orario del broker e l'orario italiano. Impostarlo correttamente è essenziale: un valore errato causa uno spostamento della fascia operativa e genera segnali negli orari sbagliati. Il valore cambia in base al broker (CME Chicago o EUREX UTC) e alla stagione
(ora legale / ora solare). The Time Zone parameter indicates the hour offset between the broker's time and Italian time. Setting it correctly is essential: a wrong value shifts the operating window and generates signals at the wrong times. The value changes based on the broker (CME Chicago or EUREX UTC) and the season
(daylight saving / standard time).
